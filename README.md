# dtmoney

> Control your expenses

## Built With

- Typescript
- React

## Setup

- Clone this repository with: `git clone https://github.com/flpfar/dtmoney.git`
- Navigate to the project folder `cd dtmoney`
- Run `yarn install`
- Run `yarn start`
- Access through `http://localhost:3000/`
